## How To Use

```bash
# Clone this repository
$ git clone https://github.com/Adewale66/alt_assignment_2.git


# change directory
$ cd assignment_7

# Install dependencies
$ npm install

# Copy and populate env variables
$ cat .env.example > .env

# Run the app
$ node index.js

# Run cron Job
$ node cron.js

```
